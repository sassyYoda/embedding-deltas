**Function 1: `sample_frames`** â€” implement the RESEARCH Gap 2 skeleton verbatim. Critical points:

- Open with `cv2.VideoCapture(str(video_path))`. If `not cap.isOpened()`, raise `RuntimeError(f"cv2.VideoCapture could not open {video_path}")`.
- `sample_interval_ms = 1000.0 / fps` (= 500.0 for fps=2.0).
- Loop: `target_ms = 0.0; while True: cap.set(cv2.CAP_PROP_POS_MSEC, target_ms); ret, frame_bgr = cap.read(); if not ret or frame_bgr is None: break`. (Pitfall 3 â€” no `CAP_PROP_FRAME_COUNT`.)
- Read timestamp **after** read: `actual_ms = cap.get(cv2.CAP_PROP_POS_MSEC)`. Accept the â‰ˆ33ms forward-bias per RESEARCH A1 â€” well under the 1.0s D-08 tolerance.
- Convert: `frame_rgb = cv2.cvtColor(frame_bgr, cv2.COLOR_BGR2RGB)` â€” **exactly once**, here, never elsewhere. (D-07, Pitfall 1.)
- Append `frame_rgb` and `actual_ms`. Increment `target_ms += sample_interval_ms`.
- Defensive infinite-loop guard: `if len(timestamps_ms) >= 2 and timestamps_ms[-1] <= timestamps_ms[-2]: break` â€” handles VFR pathologies where seek refuses to advance.
- Always `cap.release()` (use `try/finally` if you prefer; a flat call after the loop is acceptable since there's no exception path that bypasses it).
- Return `(frames, np.asarray(timestamps_ms, dtype=np.float64) / 1000.0)`. timestamps in **seconds**, float64.

Type signature: `def sample_frames(video_path: str | Path, fps: float = 2.0) -> tuple[list[np.ndarray], np.ndarray]:`.

**Function 2: `load_model`** â€” implement the RESEARCH Gap 5 skeleton verbatim. Critical points:

- Device ladder (D-11): `device = torch.device("cuda" if torch.cuda.is_available() else "mps" if torch.backends.mps.is_available() and torch.backends.mps.is_built() else "cpu")`.
- Determinism precursors (D-15): `torch.backends.cudnn.benchmark = False; torch.backends.cudnn.deterministic = True`. These are no-ops on MPS/CPU but correct on CUDA. NO env vars set here (Phase 5 owns those).
- Load: `model, _, preprocess = open_clip.create_model_and_transforms("ViT-L-14", pretrained="openai")`. Note the 3-tuple â€” `_` is `preprocess_train`, the third is `preprocess_val` which we use. (D-09, EMBD-01.)
- `model = model.to(device)`. Then `model.eval()` (D-15, Pitfall 4 â€” also fired again inside `embed_frames` defensively).
- Return `(model, preprocess, device)` exactly. No seeds set (those belong to Phase 5 entry).

Type signature: `def load_model() -> tuple[Any, Any, torch.device]:`.

**Function 3: `embed_frames`** â€” implement the RESEARCH Gap 4 skeleton verbatim. Critical points:

- Empty-input shortcut: `if not frames: return np.zeros((0, 768), dtype=np.float32)`.
- `model.eval()` at top â€” defensive, idempotent (D-15, Pitfall 4).
- Single batched loop: `for start in range(0, len(frames), batch_size): chunk = frames[start:start + batch_size]`. Trailing partial batch handled by Python slicing (no special case). (D-12, Pitfall 7.)
- Inside loop: PIL conversion `tensors = [preprocess(Image.fromarray(f)) for f in chunk]`; stack `batch_tensor = torch.stack(tensors).to(device)`. (Pitfall 5.)
- **Per-batch shape assertion** (Pitfall 5 guard): `assert batch_tensor.shape[1:] == (3, 224, 224), f"preprocess produced wrong shape: {batch_tensor.shape}"`.
- Inference: wrap the entire `for` loop in `with torch.inference_mode():` (Pitfall 4). Inside: `feats = model.encode_image(batch_tensor)` (returns (B, 768) NOT normalized â€” open_clip 3.x default).
- **Explicit L2 normalize** (D-13, Pitfall 6): `feats = feats / feats.norm(dim=-1, keepdim=True)`. Do this on-device (NOT after `.cpu().numpy()`) â€” research Don't Hand-Roll table.
- Move to numpy: `out_chunks.append(feats.cpu().numpy().astype(np.float32))`.
- After loop: `embeddings = np.concatenate(out_chunks, axis=0)`.
- **Hard L2 assertion** (D-13, EMBD-05, Pitfall 6 â€” defense in depth): `norms = np.linalg.norm(embeddings, axis=1); assert np.allclose(norms, 1.0, atol=1e-5), f"embeddings not L2-normalized: norms in [{norms.min()}, {norms.max()}]"`. This MUST fire inside `embed_frames`, not only in `__main__`.
- Return `embeddings`.

Type signature: `def embed_frames(frames: list[np.ndarray], model: Any, preprocess: Any, device: torch.device, batch_size: int = 32) -> np.ndarray:`.
